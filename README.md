# Ynov Fullstack M1 — Application de recettes de cuisine

Le projet FullStack choisit est une application qui permet aux utilsateur d'enregistrer leur propre recette de cuisine.

## Fonctionnalitées:

- Création, lecture, modification et suppression (CRUD) de recettes
- Authentification basique (inscription / connexion)
- Recherche et filtrage par ingrédients / temps de préparation
- Téléversement d'images/vidéos pour chaque recette
- Propose des recettes avec le contenue de mon frigo
- Interface responsive pour mobile et desktop


## Stack technique:

- Frontend : (ex. React, Vue ou Angular)
- Backend : Node.js + Express — API REST
- Base de données : (ex. MongoDB ou PostgreSQL)
- Stockage des images : Local ou service cloud (ex. AWS S3)
- Outils : Git, npm/yarn, VS Code
